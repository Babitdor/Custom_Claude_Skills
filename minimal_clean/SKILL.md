# Minimal Clean Design Skill

This skill provides expertise in creating minimalist and clean website designs with a focus on:
- Strategic use of whitespace
- Typography excellence
- Refined color palettes
- Content-focused layouts
- Performance optimization

## Core Principles

1. **Whitespace Mastery** - Using negative space strategically to create visual breathing room and focus attention
2. **Typography Hierarchy** - Creating clear information architecture through font sizes, weights, and spacing
3. **Color Restraint** - Using limited, harmonious palettes that enhance rather than distract from content
4. **Functional Minimalism** - Removing non-essential elements while preserving usability and clarity

## When to Use This Skill

- Creating portfolio websites
- Designing landing pages
- Improving existing website aesthetics
- Implementing clean user interfaces
- Optimizing content presentation

## Skill Components

- Design analysis and recommendations
- CSS/HTML implementation of minimalist principles
- Color palette selection
- Typography system creation
- Layout optimization techniques
